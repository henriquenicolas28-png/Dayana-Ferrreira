# Diretório de Imagens

## Estrutura de Pastas

- **usinas/** - Imagens de usinas sucroalcooleiras
- **produtores/** - Imagens de pequenos produtores e cooperativas
- **fornecedores/** - Imagens de empresas fornecedoras

## Formato Recomendado

- Formato: JPG ou PNG
- Tamanho recomendado: 800x800px ou maior
- Proporção: 1:1 (quadrado) para melhor exibição

## Nomenclatura dos Arquivos

Use nomes descritivos e sem espaços, por exemplo:
- `costa-pinto.jpg`
- `santa-barbara.jpg`
- `coop-rio-pedras.jpg`

## Placeholder

Caso uma imagem não esteja disponível, o sistema exibirá automaticamente um placeholder cinza com o texto "Sem imagem".
